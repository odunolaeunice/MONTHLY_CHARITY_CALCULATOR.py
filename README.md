# Monthly Charity Calculator 💸

This is a simple Python program I built on **Day 2** of my 100 Days of Code challenge. It helps users easily calculate how much of their monthly salary they should give to charity and how to split it evenly among multiple beneficiaries.

## 🚀 What It Does
* Takes your total monthly salary as an input.
* Asks for the percentage you want to donate (e.g., 5%, 10%, 15%).
* Asks how many people or organizations will benefit from the donation.
* Calculates the total charity amount and the exact cut each beneficiary will receive.

## 🧠 What I Learned
While building this, I practiced the following core Python concepts:
* **Data Type Conversion:** Using `float()` and `int()` to turn user text inputs into usable numbers.
* **Mathematical Operators:** Applying basic math and PEMDAS to calculate percentages.
* **Rounding Numbers:** Using `round()` to keep the final currency format clean (to two decimal places).
* **F-Strings:** Dynamically inserting calculated variables directly into print statements for a readable output.

## 💻 How to Run It
1. Ensure you have Python installed on your machine.
2. Clone this repository or download the Python file.
3. Run the script in your terminal or preferred IDE (like PyCharm or Jupyter).
4. Answer the prompts and let the code do the math for you!
